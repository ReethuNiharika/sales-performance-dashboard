# 5–10 Minute Video Walkthrough Script

Hi, my name is Reethu Niharika Peddaventhurla. This is my Sales Performance Analytics Dashboard project.

In this project, I created an end-to-end analytics workflow using Python, SQL, and an interactive dashboard. The goal of the project is to help business leaders understand sales performance, profit trends, regional performance, and top product performance.

First, the dataset is stored in the data folder as sales_orders.csv. It contains order date, region, segment, category, product, quantity, revenue, cost, profit, and customer information.

Next, I used Python for data analysis. In the src folder, the analysis.py file loads the dataset, checks the date range, calculates key metrics, and summarizes performance by region. This step helps validate the data before building the dashboard.

Then, I wrote SQL queries in the sql folder. The SQL file includes a table creation script and queries for KPI summary, monthly trends, regional performance, and product-level analysis. This shows how I would analyze the data in a real database environment.

Now I will open the dashboard. At the top, I included four key KPIs: total revenue, total profit, total orders, and profit margin. These metrics give leadership a quick understanding of business performance.

The first chart shows monthly revenue and profit trends. This helps identify seasonal patterns and months where performance improved or declined.

The second chart shows revenue by region. This helps compare geographic performance and identify where sales teams should focus.

The third chart shows revenue by category. This helps stakeholders understand which product categories contribute most to revenue.

The final chart shows top products by profit. I included profit instead of only revenue because a product can have high revenue but low profitability.

The main insight from this dashboard is that leadership can use this analysis to monitor performance, compare regions, track category contribution, and make better decisions around sales planning and discounting.
