# Sales Performance Analytics Dashboard

## Project Overview
This is an end-to-end Data Analyst project created for the Daxwell DataRev submission. The project analyzes sales order data to identify revenue trends, profit performance, regional performance, category mix, and top products.

## Tools Used
- Python: data cleaning and KPI calculation
- SQL: KPI queries and business analysis
- HTML + Plotly: interactive dashboard
- CSV dataset: sample sales orders data

## Business Questions
1. What are the total revenue, total profit, total orders, and profit margin?
2. Which regions generate the most revenue?
3. Which categories and products perform best?
4. How do revenue and profit trend month by month?
5. What insights can leadership use to improve sales planning?

## Folder Structure
```
data/sales_orders.csv
sql/sales_analysis.sql
src/analysis.py
dashboard/index.html
video_script.md
README.md
```

## How to Run
1. Open `dashboard/index.html` in a browser to view the dashboard.
2. Run `src/analysis.py` to see Python KPI analysis.
3. Use `sql/sales_analysis.sql` to recreate the analysis in a database.

## Key Insights
- Total Revenue: $1,295,804.07
- Total Profit: $473,079.94
- Total Orders: 1,800
- Profit Margin: 36.51%

## Dashboard Story
The dashboard starts with high-level KPIs, then moves into monthly revenue/profit trends, regional performance, category contribution, and top products by profit. This structure helps business leaders quickly understand performance and identify improvement areas.
